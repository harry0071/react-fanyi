## 用法
运行前请先到[官网](https://nodejs.org/zh-cn/download/)安装node.js      
然后将项目clone到本地。           
执行下面的命令

``` bash
# 首先安装依赖
yarn

# 然后运行项目
yarn start

# 打包
npm yarn build
```
